HTML Tags – Description, Syntax & Live Examples

1. Tag Name

<p>

2. Description of the tag

The <p> tag is an essential part of HTML used to define a paragraph of text within a web page. It helps structure web documents in a clear, standardised manner that browsers can correctly interpret. Using this tag improves readability, maintainability, and consistency of web pages. It is widely supported across modern browsers and plays an important role in accessibility, search engine optimisation, and proper rendering of content on different devices and screen sizes.

3. Syntax

<p>Text</p>

4. Live Example

This is a paragraph example.

5. Example Code

<p>This is a paragraph example.</p>
